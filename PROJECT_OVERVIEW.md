# 项目总览

## 📊 项目信息

**项目名称**: 智能制造生产调度系统  
**版本**: v1.0  
**开发时间**: 2025年12月  
**技术栈**: Python, Streamlit, NumPy, Pandas, Matplotlib  

## 🎯 项目目标

基于遗传算法（GA）和局部搜索（ILS/VNS）的混合优化方法，实现智能制造生产调度系统，在满足产能约束和交付期限的前提下，最大化企业利润。

## ✨ 核心功能

### 1. 智能调度算法
- **遗传算法（GA）**: 全局搜索最优生产计划
- **局部搜索（ILS/VNS）**: 精细优化调度方案
- **滚动调度**: 支持每日动态重调度

### 2. 可视化GUI
- **配置管理**: 可视化编辑算法参数
- **订单管理**: 上传CSV或使用示例数据
- **调度运行**: 一键执行调度算法
- **结果分析**: 查看甘特图和性能指标

### 3. 多维度评估
- **财务指标**: 收入、成本、罚款、利润
- **订单指标**: 完成率、按期率
- **生产指标**: 产线利用率、工作时段

## 🚀 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动GUI
```bash
streamlit run gui_app.py
```

### 命令行运行
```bash
python src/main.py
```
