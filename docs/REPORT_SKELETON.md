## 第2章 问题描述与建模

### 2.1 问题描述

**详细描述：**
- 生产线配置和能力
- 产品种类和产能参数
- 时间离散化方案（4小时/slot，6 slot/天）
- 订单属性和到达模式
- 调度时机和规则

**参考：** 《设计大纲.md》第2.1节

### 2.2 问题假设

**列出12条假设：**
1. 三条生产线能力完全相同
2. 每个时间段只能生产一种产品
3. 原材料供应充足
4. 工人能力相同
5. 人工成本按4小时结算
6. 其他成本折算进生产费用
7. 违约罚款为订单总金额的10%
8. 每日8点统一调度
9. 忽略意外情况
10. 每条产线只能生产3种产品之一或空闲
11. 截止时间为截止日期当天8点
12. 同类产品可在同一时间段合并生产

**参考：** 《设计大纲.md》第2.2节

### 2.3 符号说明

**定义符号：**

**集合与索引：**
- L = {1,2,3}：生产线集合
- P = {1,2,3}：产品集合
- T = {1,2,...,H}：时间段集合
- O = {1,2,...,N}：订单集合

**订单参数：**
- p_o：产品类型
- q_o：需求数量
- s_o：单位售价
- d_o：截止时间

**产能与成本：**
- cap_p：产品p的产能
- c_t：时间段t的人工成本

**决策变量：**
- x_{l,t}：生产线l在时间段t的产品类型
- y_{o,l,t}：订单o在(l,t)的生产数量

**参考：** 《设计大纲.md》第2.3节

### 2.4 数学模型

**目标函数：**
```
max Profit = Revenue - Cost - Penalty
```

其中：
- Revenue = ∑ q_o^done × s_o
- Cost = ∑∑ c_t × WorkIndicator_{l,t}
- Penalty = ∑ 0.1 × (q_o - q_o^done) × s_o

**约束条件：**
1. 单产品约束
2. 产能约束
3. 订单完成量约束
4. 非负约束

**参考：** 《设计大纲.md》第2.4节
